CherryTree

A toolstack for convenient development and execution of functional webscraping scripts. Scripts can be composed using a publish-subscribe like pattern.

Functional webscrapers (cherry pickers) are implemented as simple functions which are leveraged to have access to the scraping environment such that it is easy to build site traversals with decoupled scraper components.

Run these commands for initial setup and test run

$ npm install gulpjs/gulp-cli#4.0 codo -g
$ npm install
$ npm test
