Class: Crawler

Defined in: src/scraper/Crawler.coffee

Overview

The Crawler coordinates execution of submitted CrawlRequest by applying all Extensions of the ExtensionPoint that matches the request's current status.

All functionality for request handling, such as filtering, queueing, streaming, storing, logging etc. is implemented as Extensions to ExtensionPoints.

The crawler defines exactly one ExtensionPoint for each distinct value of RequestStatus. Each extension point contains the processing steps carried out when the request changes its status to the respective phase.

The processing of a request follows the RequestStatus transitions which are defined by the CrawlRequest.
