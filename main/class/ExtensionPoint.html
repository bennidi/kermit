Class: ExtensionPoint

Defined in: src/kermit/Crawler.ExtensionPoints.coffee

Overview

Provide a mechanism to add functionality to the provider of the extension point (=> Crawler) Extension points act as containers for Extensions - the primary abstraction for containment of processing functionality.
