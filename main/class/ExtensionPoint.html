Class: ExtensionPoint Abstract

Defined in: src/scraper/Crawler.coffee

Overview

This class is abstract. (An extension point should be subclassed)

An extension point provides a mechanism to add functionality to the extension point provider. Extension points act as containers for Extension - the objects providing the actual extension code
