Class: RequestStatus

Defined in: src/scraper/CrawlRequest.coffee

Overview

At any time, each request has a status value equal to one of the values defined by this class. Any request starts with status {RequestStatus.INITIAL} From status {RequestStatus.INITIAL} it transitions forward while being processed by the Extensions that handle requests of that particular status. See Crawler for a complete state diagram of the status transitions
