Class: QueueManager

Defined in: src/scraper/QueueManager.coffee

Overview

Provides access to a queue like system that allows to access CrawlRequests by their status.

Currently implemented on top of beautiful [http://lokijs.org lokijs] => Queues are emulated with dynamic views on a single request collection.
